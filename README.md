# Sistema de Barbearia

## Requisitos
Você precisa ter o **Python** instalado. 
Se ao digitar `python` no terminal nada acontecer (ou abrir a Microsoft Store), instale o Python em: https://www.python.org/downloads/

**Importante**: Na instalação, marque a opção "Add Python to PATH".

## Como Rodar

1. Abra o terminal nesta pasta (`C:\Users\Joefe\OneDrive\Arquivos Barbearia\Desktop\Nova pasta`).
2. Instale as dependências:
   ```
   pip install -r requirements.txt
   ```
3. Inicie o servidor:
   ```
   python app.py
   ```
4. Acesse no navegador:
   - **App Principal**: http://localhost:5000
   - **Painel Administrativo**: http://localhost:5000/admin

## Funcionalidades
- Crie colaboradores, serviços e produtos no Painel Admin.
- Use o botão (ou copie o link) de login do colaborador para acessar a área de vendas.
